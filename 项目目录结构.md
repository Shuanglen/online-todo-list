```text

├── index.html                    # 页面结构入口文件
├── css/
│   └── style.css                 # 页面样式、动画与响应式布局
├── js/
│   └── main.js                   # 待办事项交互逻辑与 localStorage 数据持久化
├── README.md                     # GitHub 项目说明
├── .gitignore                    # Git 忽略规则
├── Web程序设计.docx              # 课程设计报告
├── web设计模版.docx              # 课程设计报告模板
├── Web程序设计期末设计题目.pptx  # 课程期末设计题目说明
├── 查看全部事项.png              # 桌面端全部任务运行截图
├── 查看已完成事项.png            # 桌面端已完成任务筛选截图
├── 模拟手机首页.png              # 移动端首页截图
└── 模拟手机菜单栏.png            # 移动端侧边栏截图
```

## 文件说明

- `index.html`：保留 HTML 页面结构，通过 `<link>` 引入样式文件，并在页面底部引入脚本文件。
- `css/style.css`：从原 `index.html` 的 `<style>` 标签中拆分而来，包含设计变量、布局、组件样式、动画与移动端适配。
- `js/main.js`：从原 `index.html` 的 `<script>` 标签中拆分而来，包含任务添加、删除、完成状态切换、搜索筛选、统计更新、移动端菜单和本地存储逻辑。
- 运行方式：双击 `index.html`，或在 VS Code 中使用 Live Server 打开项目。
