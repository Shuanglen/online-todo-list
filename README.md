# 在线待办事项清单

《Web程序设计》课程期末作业题目二：在线待办事项清单。

## 项目简介

本项目使用 HTML5、CSS3 和原生 JavaScript 实现一个在线待办事项清单工具，支持任务新增、删除、完成状态切换、搜索筛选、优先级管理、日期显示、完成进度统计和移动端响应式布局。

## 运行方式

直接双击 `index.html` 打开，或在 VS Code 中使用 Live Server 运行。

## 主要功能

- 添加待办任务
- 删除任务并支持撤销提示
- 标记任务完成状态
- 按全部、未完成、已完成、优先级、今日任务和逾期任务筛选
- 搜索任务关键词
- 使用 localStorage 保存本地数据
- 支持桌面端和移动端响应式布局
